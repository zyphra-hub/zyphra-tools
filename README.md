## 📦 Installation

```bash
pkg install git -y && \
git clone https://github.com/zyphra-hub/zyphra-tools && \
cd zyphra-tools && \
bash install.sh
```

The installer will automatically:

- Detect your operating system
- Install Python & dependencies
- Configure the environment
